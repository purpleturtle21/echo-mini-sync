# What's New in v0.2

## Faster playlist switching
Switching between playlists is now instant after the first load. No more UI freezing.

## Track ordering fixed
Tracks now play in the correct order on Echo Mini. No more tracks 13 and 15 appearing at the end.

## Import playlists from other apps
Drop a `.m3u` file to import a playlist from MusicBee, foobar2000, or any music player that exports .m3u.

## Adopt existing folders
Already have music folders on your device? Right-click and adopt them — EchoList will manage them from now on without re-copying anything.

## Search your library
Type in the source panel to quickly find songs by name. Press Escape to clear.

## Offload playlists to save space
Right-click a playlist and choose Offload to remove files from the device while keeping the playlist structure. Onload brings them back when you need them.

## Find a track's source
Right-click any track and choose "Show in source" to jump to where it lives in your music library.

## Automatic backups
Your playlist metadata is backed up automatically. If something goes wrong, right-click any playlist to restore from a previous point.

## Better file dialogs
The folder picker now tells you what it's asking for — no more guessing which path goes where.
